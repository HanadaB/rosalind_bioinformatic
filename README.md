# Rosalind Bioinformatics Exercises
This repository contains my solutions to problems from the Rosalind Bioinformatics Project.
Rosalind is an online platform designed to teach bioinformatics and computational biology through programming challenges.
Website: https://rosalind.info

## Topics
- DNA and RNA sequence analysis
- String algorithms
- Basic genetics probability
- Bioinformatics algorithms

## Language
Most solutions are implemented in Python.

## Purpose
This repository serves as a personal collection of solutions while learning bioinformatics algorithms and computational biology.
